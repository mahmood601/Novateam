## вңЁ What is Nova App?

**Nova App** is an interactive quiz platform built specifically for medical students in Tartous university . It transforms course questions and exam materials into engaging, timed quizzes вҖ” helping students study smarter, not harder.

No more staring at static PDFs. With Nova app, your study material comes alive.

> *"Study like the exam is tomorrow. Every day."*

## рҹҡҖ Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** `>= 18`
- **pnpm** (recommended) вҖ” `npm install -g pnpm`

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/mahmood601/Novateam.git
cd Novateam

# 2. Install dependencies
pnpm install        # or: npm install / yarn install

# 3. Start the development server
pnpm dev            # or: npm run dev
```

Then open your browser at **[http://localhost:5173](http://localhost:5173)** рҹҺү

## вҳҒпёҸ Deployment

This project is deployed on **Vercel** with zero-config continuous deployment from the `master` branch.

To deploy your own fork:

1. Push your code to GitHub
2. Import the repo on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite вҖ” hit **Deploy** вң…

For other platforms, refer to the [Vite static deploy guide](https://vite.dev/guide/static-deploy.html).
